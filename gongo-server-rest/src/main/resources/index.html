GonGo PoC Demo Site

This is a proof of concept implementation of the GonGo Online Bidding application.

Functionality

  • List the exiting auctions
  • Show the details of a selected auction
  • Place a new bid on an auction
  • List the predefined products
  • Start a new auction selling a selected predefined product

There are many features that the products aim and not implemented in this PoC.

The PoC is implemented using SpringBoot, HTML, JavaScript and CSS.

The persistence layer uses flat files in yaml format.

Development time 12h. Design time 7h.
